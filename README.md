# Pie
Makes an array of pies


## Description
This demonstrates using the DOM and onClick events

## Getting Started
```
$ git clone https://github.com/wchipchase/pie/
```
### Prerequisites
```
$ npm install -g http-server
```

## Running
Browse to pie directory and run http server
